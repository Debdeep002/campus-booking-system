# 🏫 Campus Room Booking System

A full-stack, 3-tier web application built for seamless campus room management and booking. 

🚀 **Live Demo:** https://campus-booking-system-taupe.vercel.app

### 🏗️ Architecture
* **Frontend:** Vanilla HTML/CSS/JS (Hosted on Vercel)
* **Backend:** Java Spring Boot (Hosted on Render)
* **Database:** MySQL (Hosted on TiDB Cloud)
